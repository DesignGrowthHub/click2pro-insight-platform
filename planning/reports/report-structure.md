# Report Structure

## Report Goal

Convert assessment responses into a premium-feeling, psychologically intelligent insight product with a clear difference between free and paid value.

## Free Snapshot

- Short pattern label
- 2 to 3 insight bullets
- One high-level explanation
- One CTA to unlock the full report

## Paid Report Sections

1. Core pattern summary
2. What is likely driving this pattern
3. How it affects relationships, mood, or self-perception
4. Common triggers and amplifiers
5. Strengths hidden inside the pattern
6. Blind spots and risks
7. Practical reflection prompts
8. Recommended next assessment or bundle path

## Tone Rules

- Serious and calm
- Emotionally precise
- Personalized without sounding clinical
- Insight-oriented, not diagnostic

## Assembly Notes

- Reports should be modular so each assessment can reuse a shared structure.
- Variables should support scoring bands, trait emphasis, and cross-sell recommendations.
- The report should feel deeper than a generic score readout.
