# Toxic Pattern & Red Flag Report

## User Problem

The user suspects an unhealthy relationship dynamic and wants clearer language around manipulation, instability, disrespect, or recurring harm.

## Entry Topics

- Toxic patterns
- Red flags
- Manipulation

Signal whether the pattern suggests mild incompatibility, escalating instability, or consistent red-flag behavior.

## Paid Report Angle

Interpret specific relational warning signs, likely pattern cycles, and boundary pressure points.

## Key Dimensions

- Consistency of respect
- Manipulation indicators

- Emotional volatility
