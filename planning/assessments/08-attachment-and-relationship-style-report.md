# Attachment & Relationship Style Report

## User Problem

The user wants clarity on the relational style they keep repeating and how that style shapes closeness, trust, pursuit, withdrawal, and conflict.

## Entry Topics

- Attachment style
- Avoidant behavior
- Anxious attachment
- Relationship patterns

## Free Result Angle

Show whether the user's pattern leans anxious, avoidant, mixed, or relatively secure under stress.

## Paid Report Angle

Detail triggers, conflict habits, reassurance needs, distancing patterns, and compatibility implications.

## Key Dimensions

- Closeness tolerance
- Reassurance need
- Conflict response
- Pursuit vs withdrawal
- Trust sensitivity
