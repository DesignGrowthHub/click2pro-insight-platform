# Relationship Infatuation / Obsession Analysis

## User Problem

The user feels mentally consumed by another person and wants to understand whether the attachment is healthy desire, limerence, fantasy projection, or emotional dependence.

## Entry Topics

- Infatuation
- Obsession
- Limerence
- Over-attachment

## Free Result Angle

Show whether the attachment pattern is based more on fantasy intensity, intermittent reinforcement, unmet needs, or anxious fixation.

## Paid Report Angle

Detail the user's attachment drivers, obsession loops, emotional vulnerability points, and grounding strategies.

## Key Dimensions

- Thought intrusiveness
- Fantasy vs reality gap
- Emotional dependency
- Reciprocation ambiguity
- Self-regulation capacity
