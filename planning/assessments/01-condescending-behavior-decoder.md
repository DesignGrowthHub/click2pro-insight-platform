# Condescending Behavior Decoder

## User Problem

The user feels talked down to, subtly belittled, or mentally minimized and wants help interpreting whether the pattern is real, repeated, and meaningful.

## Entry Topics

- Condescending behavior
- Patronizing communication
- Being treated as inferior

## Free Result Angle

Show whether the user is likely dealing with isolated friction, repeated status signaling, or a more stable disrespect pattern.

## Paid Report Angle

Break down behavior cues, likely dynamics, emotional impact, and boundary options.

## Key Dimensions

- Frequency and context
- Power imbalance
- Public vs private behavior
- Emotional aftereffect
- Boundary viability
