# Closure & Emotional Recovery Report

## User Problem

The user is trying to move on from a person, dynamic, or emotional chapter and wants to understand what is still psychologically unresolved.

## Entry Topics

- Closure
- Letting go
- Emotional recovery
- Still not over it

## Free Result Angle

Show whether the user is blocked more by unanswered meaning, attachment residue, identity disruption, or hope that has not fully collapsed.

## Paid Report Angle

Interpret the user's recovery stage, unfinished loops, and the emotional threads keeping the past active.

## Key Dimensions

- Attachment residue
- Rumination intensity
- Meaning-seeking
- Identity after-loss
- Readiness to release
