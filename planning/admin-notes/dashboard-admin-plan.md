# Dashboard / Admin Plan

## Admin Goal

Give the internal team enough control to manage assessments, mappings, pricing, reports, and conversion monitoring without turning the first release into a heavy internal software project.

## Minimum Admin Needs

- Manage assessment metadata and status
- Manage blog-topic-to-assessment mappings
- Update pricing, bundles, and subscription offers
- Review report templates and recommended cross-sells
- View funnel metrics by topic and assessment

## Suggested Admin Sections

1. Assessments
2. Blog routing map
3. Report templates
4. Offers and pricing
5. Orders and subscriptions
6. Analytics overview

## Launch Note

For the first build, lightweight internal forms or direct data-table editing may be enough. A polished admin product can come later if the funnel validates.
